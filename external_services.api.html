external_services.api.oauth_nonce_cache module

class external_services.api.oauth_nonce_cache.OAuthNonceCache(nonce, timestamp, client_secret)[source]

Bases: object

Cache for OAuth nonce values so that a received nonce may be checked against previously seen nonces. In OAuth1, the nonce value MUST be unique across all requests with the same timestamp, client credentials, and token combinations. Since LTI 1.1 does not use OAuth tokens, tokens are ignored here. Uses the default cache configured in Django.

CACHE_TIMEOUT_SECONDS = 600
nonce_used()[source]
