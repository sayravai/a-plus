class deviations.models.SubmissionRuleDeviation(*args, **kwargs)[source]

Bases: UrlMixin, Model

An abstract model binding a user to an exercise stating that there is some kind of deviation from the normal submission boundaries, that is, special treatment related to the submissions of that particular user to that particular exercise.

If there are many submitters submitting an exercise out of bounds of the default bounds, all of the submitters must have an allowing instance of SubmissionRuleDeviation subclass in order for the submission to be allowed.
